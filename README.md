# TP Movies PHP - Niveau 4 terminé

## Description
Ce projet est un gestionnaire de films qui récupère des données via des fichiers HTML locaux (scraping) et les stocke dans une base de données MySQL.

## Fonctionnalités
- **Niveau 1** : Extraction du titre, de l'affiche, de la note et des acteurs via Regex.
- **Niveau 2** : Connexion PDO et enregistrement des avis en base de données.
- **Niveau 3** : Affichage de la collection personnelle sur la page d'accueille.
- **Niveau 4** : Moteur de recherche basé sur les fichiers de recherche fourniis.

## Installation
1. Importer le fichier `tp_movies.sql` dans votre interface phpMyAdmin.
2. Configurer les identifiants de connexion dans `config.php` (actuellement configuré pour MAMP : root/root).
3. Placer le dossier dans `htdocs` et accéder via `localhost:81/TP-Movies-PHP/`.
